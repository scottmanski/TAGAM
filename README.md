# TAGAM

<!-- badges: start -->

<!-- badges: end -->

The goal of TAGAM is to streamline the use of word embeddings in text
analysis. The package is designed to take raw textual descriptions and
transform the information for use in generalized additive models. The
package also includes functions for running the GAM framework described
in XXX.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("scottmanski/TAGAM")
```
